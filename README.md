# Carbon Footprint Tracker

A web app to track and reduce your carbon footprint.

## Features
- Log and track daily activities
- Calculate carbon emissions
- Receive eco-friendly suggestions
- Visualize carbon footprint data

## Getting Started
### Prerequisites
- Node.js
- Firebase account

### Installation
